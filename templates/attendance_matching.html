Match attendance data from different sources using intelligent algorithms

Match attendance data from different sources using intelligent name matching.
Upload master file(s) containing participant names and email addresses.
Upload raw attendance file(s) to match with master files.
Important Instructions
  • File Pairing: The first master file will be matched with the first raw file, the second master with the second raw, and so on.
  • Selection Order: Please ensure files are selected in the correct order.
  • Equal Numbers: You should select the same number of master and raw files.
Instructions
How to Use This Tool
  1. Select Master Files: Choose one or more master files containing participant names and email addresses
  2. Select Raw Files: Choose the corresponding raw attendance files to match
  3. File Pairing: Files are matched in order:
    • 1st master file ↔ 1st raw file
    • 2nd master file ↔ 2nd raw file
    • And so on...
  4. Choose Output Format: Select Excel (.xlsx) or CSV format for results
  5. Process: Click "Match Attendance" to start processing
Example

If you have:

  • Master files: ClassA_Master.xlsx, ClassB_Master.xlsx
  • Raw files: ClassA_Attendance.csv, ClassB_Attendance.xlsx

The tool will match:

  • ClassA_Master.xlsx with ClassA_Attendance.csv
  • ClassB_Master.xlsx with ClassB_Attendance.xlsx

This tool uses fuzzy matching algorithms to match attendance records from different sources, even when names are spelled differently or formatted differently.
